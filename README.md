מטרת המשחק:
<div dir='ltr'>

	אנחנו משחקים בתור החללית למטה. אנו יכולים לירות לייזר ולחסל את החלליות שמולנו. ככל שנחסל יותר חלליות כך נקבל יותר ניקוד. כאשר אנו מתנגשים בחללית אוייבת יורד לנו חיים. לחללית שלנו 3 חיים סך הכל.

</div>

מקשים:
<div dir='ltr'>

	right arrow - right, left arrow - left, up arrow - up, down arrow - down and space to shoot laser.

</div>
